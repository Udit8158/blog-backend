## API Docs

# Auth
POST localhost:80/api/v1/auth/register - sign up
POST localhost:80/api/v1/auth/login - sign in
POST localhost:80/api/v1/auth/logout - sign out

# Blogs
GET localhost:80/api/v1/admin/blogs?blog={numberOfBlogs} - get all blogs (number of blogs)
POST localhost:80/api/v1/admin/blogs - create a blog
